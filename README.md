# 🌱 AgroBalance

AgroBalance é um site interativo que simula decisões no agronegócio e seus impactos ambientais. O projeto mostra em tempo real produção, sustentabilidade e impacto, com recomendações, histórico e modo claro/escuro. Feito com HTML, CSS e JavaScript, une educação e consciência ambiental.

---

## 🎯 Objetivo do Projeto

O objetivo do AgroBalance é demonstrar, de forma simples e interativa, como escolhas no agronegócio influenciam diretamente o meio ambiente e a produção. O projeto busca incentivar a reflexão sobre práticas sustentáveis por meio de simulações visuais e feedback imediato.

---

## ⚙️ Recursos Técnicos

- HTML5 semântico para estrutura do site  
- CSS3 com responsividade e modo claro/escuro  
- JavaScript puro para lógica do simulador  
- Armazenamento local (localStorage) para histórico  
- Atualização dinâmica de resultados em tempo real  

---

## ♿ Hub de Acessibilidade

- Aumento e diminuição de fonte  
- Leitura em voz alta da página  
- Modo claro e modo escuro  
- Interface adaptada para melhor leitura  

---

## 🎨 UI / UX e Design

O design foi pensado para ser limpo, intuitivo e educativo. O uso de cards, cores naturais e animações leves ajuda na compreensão dos dados e melhora a experiência do usuário. O foco é tornar a navegação simples e agradável, mantendo a atenção no conteúdo educativo.
